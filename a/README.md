# NOIRÉ — Demo Fashion Homepage

静的ファイルだけで作った「おしゃれなファッションサイト風」トップページのデモです（UI盛り盛り）。

## 使い方

1. `index.html` をブラウザで開くだけで動きます。
2. ショートカット
   - `Ctrl + K`: 検索モーダル
   - `Esc`: モーダル/ドロワーを閉じる

## 構成

- `index.html`: ページ本体（セクション/モーダル/ドロワー）
- `styles.css`: デザイン（ガラス調、カード、アニメーション、レスポンシブ）
- `script.js`: 体験（メガメニュー、検索、クイックビュー、カート、トースト等）

※ デモなので決済や検索結果などのバックエンドはありません。
